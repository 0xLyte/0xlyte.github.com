Black Hat USA 2019 - Breaking Samsung's ARM TrustZone

Using Samsung's TrustZone implementation as a target, this presentation explains and demonstrates how this new attack surface can be leveraged to hijack and exploit trusted components.
After explaining the internals and interactions of these components developed by Samsung, different vulnerabilities are detailed and exploited to execute code at EL3, the highest privilege level on an ARM-based system.

Quarkslab - Security Audit of Particl's Bulletproof and MLSAG

Security assessment of the non-interactive zero-knowledge proof Bulletproof and the ring signature protocol MLSAG implemented in the Particl cryptocurrency.
These systems are used in cryptocurrency transactions to ensure that they do not leak the amount exchanged and the exact identity of the actors involved.

Report

MISC - CVE-2017-6862: How I Routed Your Router

Publication in the french security magazine MISC about a vulnerability found in a Netgear router that can be exploited to achieve code execution remotely.
It presents a simple methodology to start looking for vulnerabilities in MIPS-based embedded devices (SOHO routers in particular) and exploit them.
